# AI Notes AWS Project

This is a serverless AWS project that allows users to upload notes/files to S3, receive email notifications via SNS, and log data using Lambda and DynamoDB.

## Services Used

- AWS Lambda
- Amazon S3
- Amazon SNS
- IAM
- Amazon DynamoDB
